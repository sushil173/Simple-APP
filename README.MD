# REACT NATIVE SIMPLE APP

### RUN

```
1. CLONE
2. RUN 'yarn' or 'yarn install'
3. RUN 'cd ios' and 'pos install' (For iOS)
4. react-native run-ios (For iOS) / react-native run-android ( For Android )
5. Fun!
```
